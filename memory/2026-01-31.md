## 信息分析能力自我提升（重要里程碑）

用户要求我**主动提升自己的信息分析能力**，已创建完整的自我提升系统：

### 创建的自动化工具

| 工具 | 文件 | 功能 |
|------|------|------|
| **关键词生成器** | `keyword-generator.sh` | 根据用户需求自动生成多组搜索词（中英文、技术/平台组合） |
| **GitHub评分器** | `github-scorer.sh` | 根据 star数、更新、fork、issues 自动评分（满分100） |
| **信息验证器** | `info-validator.sh` | 权威性、时效性、交叉验证清单 |
| **自我诊断** | `self-diagnosis.md` | 分析当前5个弱项：搜索受限、自动化低、跨平台弱、验证缺失、无关键词工具 |
| **改进计划** | `improvement-plan.md` | 短/中/长期提升路径 |

### 工具使用示例

```bash
# 生成搜索词
./keyword-generator.sh "知乎 爬虫"
# 输出: zhihu crawler python, 知乎 爬虫 github 等

# 评估项目质量
./github-scorer.sh 500 30 100 10
# 输出: 总分/100 + 质量等级

# 验证信息
./info-validator.sh "github.com" "2026-01-31" "作者"
# 输出: 权威性、时效性、置信度评估
```
