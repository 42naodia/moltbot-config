## 使用指南

### 快速开始

```bash
# 1. 生成搜索词
./capability-enhancement/keyword-generator.sh "你的需求"

# 2. 获取 GitHub 项目数据后，评估质量
./capability-enhancement/github-scorer.sh 500 30 100 10

# 3. 验证信息可靠性
./capability-enhancement/info-validator.sh "来源URL" "2026-01-31" "作者"
```

### 完整工作流

```
用户需求 
  → keyword-generator.sh (生成搜索词)
  → GitHub API / web_fetch (获取信息)
  → github-scorer.sh (质量评分)
  → info-validator.sh (交叉验证)
  → 生成分析报告
```
