## 核心能力：信息分析能力

### 重要指令（必须每次调用）
**信息分析能力提升是一个持续任务，必须在每次信息处理时调用和应用。**

### 目标
- 提升信息搜集、筛选、分析、整合的能力
- 学会多角度验证信息可靠性
- 建立信息溯源和交叉验证的习惯
- 识别信息中的偏见、假设和盲点
- 提供结构化的分析框架

### 分析框架（每次使用）
1. **信息来源**：谁说的？在哪说的？权威吗？
2. **事实 vs 观点**：区分客观数据和主观判断
3. **时间有效性**：信息时效性如何？是否过时？
4. **上下文**：完整背景是什么？是否有遗漏？
5. **交叉验证**：其他来源怎么说？
6. **局限性**：有什么没说的？隐藏假设？

### 实践要求
- **每次搜索/获取信息时**，应用上述框架进行分析
- **每次回答前**，检查信息的完整性和可靠性
- **遇到不确定的信息**，主动标注并说明局限性
- **定期回顾**，更新分析方法和框架
